### <BucketList 화면><br>
입력한 후 +버튼을 눌러 버킷리스트를 추가한다 <br>
삭제하고 싶을 때는 버킷리스트를 누른 이후에 – 버튼을 누른다.<br>
저장하고싶으면 save 버튼을 누르고 앱을 나갔다 들어왔을 때는 load 버튼을 눌러 이전에 작성했던 것을 불러온다.<br>
